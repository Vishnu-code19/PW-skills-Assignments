	2.Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML document?
	Ans: The purpose of comments in HTML is to put an explanatory note in the code that conveys a certain message or explanation to the person interacting with source code.
	Example: <!-- This is a comment --> 
	Note : A comment is only visible in source code it would not get rendered on the browser.
