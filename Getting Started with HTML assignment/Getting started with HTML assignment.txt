	1.	Write a simple program in HTML that displays the heading “HTML defines the content and structure of your website” on a web browser?
	Ans:<h1>HTML defines the content and structure of your website</h1>

	2.	Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML document?
	Ans: The purpose of comments in HTML is to put an explanatory note in the code that conveys a certain message or explanation to the person interacting with source code.
	Example: <!-- This is a comment --> 
	Note : A comment is only visible in source code it would not get rendered on the browser.

	4.	Write a short note on Tag and element with an example?
	Ans:Tag is a building block of html consisting of angle brackets. e.g., <h1>
	    Element is a combination of opening tag and closing tag with content. e.g., <h1>content</h1>

	5.	What is the doctype declaration in HTML?
	Ans:Doctype declaration is instruction to browser that conveys the document type is html and its version.
