HTML Tags

  1. <!DOCTYPE> ==> Defines the document type

  2. <a> ==> Defines a hyperlink

  3. <abbr> ==> Defines an abbreviation or an acronym

  4. <address> ==> Defines contact information for the author/owner of a document

  5. <applet> ==> Not supported in HTML5. Use <embed> or <object> instead.

    6. Defines an embedded applet

  6. <area> ==> Defines an area inside an image map

  7. <article> ==> Defines an article

  8. <aside> ==> Defines content aside from the page content

  9. <audio> ==> Defines embedded sound content

  10. <b> ==> Defines bold text

  11. <base> ==> Specifies the base URL/target for all relative URLs in a document

  12. <basefont> ==> Not supported in HTML5. Use CSS instead.

    13. Specifies a default color, size, and font for all text in a document

  13. <bdi> ==> Isolates a part of text that might be formatted in a different direction from other text outside it

  14. <bdo> ==> Overrides the current text direction

  15. <big> ==> Not supported in HTML5. Use CSS instead.

    16. Defines big text

  16. <blockquote> ==> Defines a section that is quoted from another source

  17. <body> ==> Defines the document's body

  18. <br> ==> Defines a single line break

  19. <button> ==> Defines a clickable button

  20. <canvas> ==> Used to draw graphics, on the fly, via scripting (usually JavaScript)

  21. <caption> ==> Defines a table caption

  22. <center> ==> Not supported in HTML5. Use CSS instead.

    23. Defines centered text

  23. <cite> ==> Defines the title of a work

  24. <code> ==> Defines a piece of computer code

  25. <col> ==> Specifies column properties for each column within a element

  26. <colgroup> ==> Specifies a group of one or more columns in a table for formatting

  27. <data> ==> Adds a machine-readable translation of a given content

  28. <datalist> ==> Specifies a list of pre-defined options for input controls

  29. <dd> ==> Defines a description/value of a term in a description list

  30. <del> ==> Defines text that has been deleted from a document

  31. <details> ==> Defines additional details that the user can view or hide

  32. <dfn> ==> Specifies a term that is going to be defined within the content

  33. <dialog> ==> Defines a dialog box or window

    34. Defines a directory list

  34. <div> ==> Defines a section in a document

  35. <dl> ==> Defines a description list

  36. <dt> ==> Defines a term/name in a description list

  37. <em> ==> Defines emphasized text

  38. <embed> ==> Defines a container for an external application

  39. <fieldset> ==> Groups related elements in a form

  40. <figcaption> ==> Defines a caption for a <figure> element

  41. <figure> ==> Specifies self-contained content

  42. <font> ==> Not supported in HTML5. Use CSS instead.

    43. Defines font, color, and size for text

  43. <footer> ==> Defines a footer for a document or section

  44. <form> ==> Defines an HTML form for user input

  45. <frame> ==> Not supported in HTML5.

    46. Defines a window (a frame) in a frameset

  46. <h1> to <h6> ==> Defines HTML headings

  47. <head> ==> Contains metadata/information for the document

  48. <header> ==> Defines a header for a document or section

  49. <hr> ==> Defines a thematic change in the content

  50. <html> ==> Defines the root of an HTML document

  51. <i> ==> Defines a part of text in an alternate voice or mood

  52. <iframe> ==> Defines an inline frame

  53. <img> ==> Defines an image

  54. <input> ==> Defines an input control

  55. <ins> ==> Defines a text that has been inserted into a document

  56. <kbd> ==> Defines keyboard input

  57. <label> ==> Defines a label for an <input> element

  58. <legend> ==> Defines a caption for a <fieldset> element

  59. <li> ==> Defines a list item

  60. <link> ==> Defines the relationship between a document and an external resource (most used to link to style sheets)

  61. <main> ==> Specifies the main content of a document

  62. <map> ==> Defines an image map

  63. <mark> ==> Defines marked/highlighted text

  64. <meta> ==> Defines metadata about an HTML document

  65. <meter> ==> Defines a scalar measurement within a known range (a gauge)

  66. <nav> ==> Defines navigation links

  67. <noscript> ==> Defines an alternate content for users that do not support client-side scripts

  68. <object> ==> Defines a container for an external application

  69. <ol> ==> Defines an ordered list

  70. <optgroup> ==> Defines a group of related options in a drop-down list

  71. <option> ==> Defines an option in a drop-down list

  72. <output> ==> Defines the result of a calculation

  73. <p> ==> Defines a paragraph

  74. <param> ==> Defines a parameter for an object

  75. <picture> ==> Defines a container for multiple image resources

  76. <pre> ==> Defines preformatted text

  77. <progress> ==> Represents the progress of a task

  78. <q> ==> Defines a short quotation

  79. <rp> ==> Defines what to show in browsers that do not support ruby annotations

  80. <rt> ==> Defines an explanation/pronunciation of characters (for East Asian typography)

  81. <ruby> ==> Defines a ruby annotation (for East Asian typography)

  82. <s> ==> Defines text that is no longer correct

  83. <samp> ==> Defines sample output from a computer program

  84. <script> ==> Defines a client-side script

  85. <section> ==> Defines a section in a document

  86. <select> ==> Defines a drop-down list

  87. <small> ==> Defines smaller text

  88. <source> ==> Defines multiple media resources for media elements (<video> and <audio>)

  89. <span> ==> Defines a section in a document

  90. <strong> ==> Defines important text

  91. <style> ==> Defines style information for a document

  92. <sub> ==> Defines subscripted text

  93. <summary> ==> Defines a visible heading for a <details7gt; element

  94. <sup> ==> Defines superscripted text

  95. <svg> ==> Defines a container for SVG graphics

  96. <table> ==> Defines a table

  97. <tbody> ==> Groups the body content in a table

  98. <td> ==> Defines a cell in a table

  99. <template> ==> Defines a container for content that should be hidden when the page loads

  100. <textarea> ==> Defines a multiline input control (text area)

  101. <tfoot> ==> Groups the footer content in a table

  102. <th> ==> Defines a header cell in a table

  103. <thead> ==> Groups the header content in a table

  104. <time> ==> Defines a specific time (or datetime)

  105. <title> ==> Defines a title for the document

  106. <tr> ==> Defines a row in a table

  107. <track> ==> Defines text tracks for media elements

  108. <tt> ==> Not supported in HTML5. Use CSS instead.
    109. Defines teletype text

  109. <u> ==> Defines some text that is unarticulated and styled differently from normal text

  110. <ul> ==> Defines an unordered list

  111. <var> ==> Defines a variable

  112. <video> ==> Defines embedded video content

  113. <wbr> ==> Defines a possible line-break
